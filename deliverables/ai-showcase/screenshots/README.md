## Nên chụp thêm gì? (tùy chọn)

Chọn **1 trong 2** nếu muốn bổ sung:

### 1. Competitive hoặc Skill (research + số liệu có nguồn)

Chụp một đoạn chat Cursor trong đó có **cả hai**:
- **Search / web:** Kết quả search (vd "ClawHub OpenClaw skill registry stats" hoặc "DeBank portfolio users") — hoặc bạn gõ search query và AI trả về số liệu + nguồn.
- **Prompt Cursor:** Câu bạn gửi cho Cursor (vd "Viết block Đối thủ 1 ClawHub..." hoặc "Viết Skill 2.2 Portfolio Tracker...") và phần đầu câu trả lời.

→ Thể hiện: không chỉ "AI cho em" mà có **search → verify → rồi mới đưa vào deliverable**.

### 2. Distribution (4 lợi ích + phân bổ ít/free)

Chụp đoạn chat có prompt kiểu:
- *"Viết 4 lợi ích kép từ cuộc thi: (1) Nguồn skill từ người dự thi (2) FOMO (3) Nguồn tester vô tận (4) Kéo user từ skill market khác..."*
- Hoặc *"Viết section Các hướng khác: phân bổ ít hoặc free cho KOL, cộng đồng, Ads... và lý do từng dòng."*

→ Thể hiện: distribution plan là **thảo luận với AI** (prompt rõ, output map vào distribution-plan.md).
