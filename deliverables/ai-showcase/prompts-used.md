## 1. Competitive Landscape (Bức tranh cạnh tranh)

**Mục tiêu:** Tìm 5–10 đối thủ (Web3 skill / agent marketplace), có số liệu, phân tích điểm mạnh/yếu, kết luận ClawFriend khác ở đâu.

**Prompt / workflow đã dùng:**

- **Prompt 1:** *"Web3 AI agent marketplace skill plugin 2024 2025"* — Tìm platform: Heurist Mesh, SkillGem, ClawMarket, Hedera Agent Marketplace, v.v.
- **Prompt 2:** *"ClawHub OpenClaw skill registry stats users"* — Lấy số liệu: 5.700+ skills, 180K+ GitHub stars, top skill downloads.
- **Prompt 3:** *"friend.tech bonding curve users TVL 2024 2025"* — Số liệu: 913K traders, TVL, volume (DeFi Llama, Dune).
- **Prompt 4:** *"Virtuals Protocol AI agent tokenization"*, *"SingularityNET AGIX AI marketplace"*, *"ELIZA ai16z AI agent framework GitHub"* — Chi tiết từng đối thủ, monetization, điểm mạnh/yếu.
- **Prompt 5:** Yêu cầu tổng hợp thành bảng đối thủ + phân tích từng cái + kết luận định vị ClawFriend (khác ở đâu, thắng/thua).

**Công cụ:** Web search + Cursor (soạn competitive-landscape.md từ kết quả, có ghi nguồn).

## 2. Skill Research (Nghiên cứu skill)

**Mục tiêu:** 5–10 skill, mỗi skill có target user, problem, alternative, bằng chứng demand, visibility/monetization.

**Prompt / workflow đã dùng:**

- **Prompt 1:** *"whale alert crypto tracking demand Nansen Arkham pricing users"* — Nansen $99–$999/tháng, Whale Alert 2.5M followers.
- **Prompt 2:** *"rug pull scam token detector crypto tool demand 2024"* — RugChecker, TokenSpy, Token Sniffer, demand đã validate.
- **Prompt 3:** *"DeFi yield optimizer APY tool demand users"* — YO Protocol, maxAPY, nhu cầu so sánh yield.
- **Prompt 4:** *"token price alert Telegram bot crypto demand"*, *"Whale Alert Twitter followers crypto"* — Các tool/bot hiện có, số liệu demand.
- **Prompt 5:** Yêu cầu viết 5–7 skill (tên, target user, problem, alternative, skill giải quyết thế nào, visibility & monetization, bằng chứng demand có nguồn) — align với BSC, ClawFriend (holder-gated).

**Công cụ:** Web search + Cursor (soạn skill-research.md, mỗi skill có nguồn).

## 3. Distribution Plan (Kế hoạch phân phối)

**Mục tiêu:** Kế hoạch $10K, không KOL/community chung chung, đột phá; sau đó tập trung chỉ cuộc thi hàng tuần.

**Prompt / workflow đã dùng:**

- **Prompt 1:** Yêu cầu ý tưởng distribution đột phá, không thuê KOL, không "làm cộng đồng" chung — → các ý: Agent-as-KOL, skill trojan ClawHub, wedge, bot Telegram, Creator = agent owner.
- **Prompt 2:** User đưa ý tưởng cuộc thi hàng tuần: BGK mua $2.5K share winner, BXH realtime, đề bài skill, FOMO mua sớm chờ BGK đẩy giá. Yêu cầu đánh giá và viết lại distribution plan xoay quanh cuộc thi.
- **Prompt 3:** Yêu cầu bỏ các kênh khác, ghi rõ "đã cũ, thí sinh khác làm, không đột phá", chỉ giữ cuộc thi.
- **Prompt 4:** Yêu cầu thêm lợi ích kép: nguồn skill chất lượng từ người dự thi, lý do FOMO (mua sớm share khi rẻ chờ BGK đẩy giá), nguồn tester vô tận (phải cài skill để đánh giá).
- **Prompt 5:** Yêu cầu cơ chế BXH: T2–T7 update realtime, Chủ nhật không update (đóng băng) để tránh đổ xô mua gần cuối, user hồi hộp ngày cuối.

**Công cụ:** Cursor (đọc guidebook, spec, thảo luận ý tưởng với user, viết và chỉnh distribution-plan.md).
